## 截图要求

- 格式：PNG
- 分辨率：建议 1920x1080 或更高
- 内容：确保关键按钮/输入框清晰可见
- 语言：中文界面

## 如何截图

### Windows
- `Win + Shift + S` 打开截图工具
- 或使用 `Print Screen` 键截取全屏

### macOS
- `Cmd + Shift + 4` 截取选定区域
- `Cmd + Shift + 3` 截取全屏

### 浏览器
- `F12` 打开开发者工具
- 点击设备工具栏图标切换响应式视图
- 右键 → 截图

## 截图完成后

将截图文件放入此目录，GUIDE.md 中的图片链接会自动生效。
