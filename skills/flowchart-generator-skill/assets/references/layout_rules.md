# 布局规则与最佳实践

---

## 1. 流向选择策略

| 条件 | 推荐流向 | 原因 |
|------|---------|------|
| 步骤数 ≤ 10，有分支 | TB（自上而下） | 分支左右展开，直观清晰 |
| 步骤数 > 10，线性为主 | LR（从左到右） | 避免纵向过长 |
| 大量并行分支 | TB | 并行横向排列更自然 |
| 泳道/角色区分明显 | LR | 泳道纵向分隔，横向流转 |

**默认**：除非有明确理由，一律使用 TB。

---

## 2. 网格定位规则

### TB 模式（自上而下）

```
        col=0      col=1      col=2      col=3
row=0              [开始]
row=1              [步骤1]
row=2              {判断?}
row=3   [否路径]              [是路径]
row=4   [处理A]               [处理B]
row=5              [汇合]
row=6              [结束]
```

**规则**：
- 主流程走 col=1（居中列）
- 分支向 col=0（左）和 col=2（右）展开
- 二次分支可用 col=3 或更多列
- 汇合节点回到主流程列
- 同层级节点在同一 row

### LR 模式（从左到右）

```
        row=0     [开始] → [步骤1] → {判断?} → [是路径] → [结束]
        row=1                           ↓
                                    [否路径] → [处理] ──→ ↑
```

**规则**：
- 主流程走 row=0
- 分支向 row=1, row=2 展开
- 回路通过底部绕回

## 3. 间距规范

| 参数 | TB 模式 | LR 模式 | 说明 |
|------|---------|---------|------|
| node_gap_vertical | 70px | 50px | 节点垂直间距 |
| node_gap_horizontal | 60px | 80px | 节点水平间距 |
| canvas_padding | 50px | 50px | 画布内边距 |
| title_margin_bottom | 30px | 30px | 标题与内容间距 |
| footnote_margin_top | 30px | 30px | 内容与脚注间距 |

---

## 4. 节点尺寸规范

| 节点类型 | 最小宽度 | 高度 | 备注 |
|---------|---------|------|------|
| terminal | 120px | 44px | 胶囊形 |
| process | 150px | 52px | 圆角矩形 |
| decision | 120px | 70px | 菱形（宽×高） |
| io | 150px | 52px | 平行四边形 |
| subprocess | 150px | 52px | 双边框矩形 |

**自适应规则**：
- 节点宽度 = max(min_width, 文字宽度 + padding_h × 2)
- 同类型节点取最大宽度统一（保持视觉整齐）
- 同一行的节点高度统一

---

## 5. 连接线路由规则

### 优先级策略

1. **直线优先**：同列上下相邻节点用直线连接
2. **单折线次之**：不同列节点用 L 形折线（先垂直再水平，或先水平再垂直）
3. **双折线兜底**：需要绕过障碍时用 Z 形双折线

### 避免交叉

- 调整分支节点的左右位置，使连接线不交叉
- 如无法避免交叉，使主流程连线在上层（先渲染分支线）

### 回路处理

- 循环回路从目标节点的左侧绕行
- 使用虚线或不同颜色区分回路

---

## 6. 视觉层次

```
最底层    画布背景
   ↑      分组/泳道背景
   ↑      连接线
   ↑      节点图形（含阴影）
   ↑      节点文字
   ↑      连接线标签（带白色背景遮挡）
最顶层    标题 / 脚注 / 注释
```

## 7. 标题与备注布局

```
┌──────────────────────────────────────────┐
│  (padding)                               │
│         流程标题（居中，大号粗体）          │
│         副标题（居中，小号灰色）            │
│  (title_margin_bottom)                   │
│                                          │
│         ┌──────────────┐                 │
│         │   流程内容    │                 │
│         └──────────────┘                 │
│                                          │
│  (footnote_margin_top)                   │
│         脚注/备注（居中，小号斜体灰色）     │
│  (padding)                               │
└──────────────────────────────────────────┘
```

---

## 8. 常见反模式（避免）

| ❌ 反模式 | ✅ 正确做法 |
|-----------|------------|
| 节点大小不一致 | 同类型节点统一尺寸 |
| 间距不均匀 | 使用网格对齐 |
| 连接线斜着穿越多个节点 | 使用正交折线 |
| 判断分支没有标签 | 每条分支必须标注条件 |
| 文字溢出节点 | 缩短标签或自动折行 |
| 颜色过多 | 限制在配色方案的 3-5 种色 |
| 阴影过重 | 使用轻阴影 (blur ≤ 6, opacity ≤ 0.1) |
