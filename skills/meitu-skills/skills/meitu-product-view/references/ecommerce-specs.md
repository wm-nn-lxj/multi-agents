# 电商平台商品图规格

Agent 在用户提到具体平台时加载此文件，用于自动选择 ratio、size 和推荐角度。

---

## 淘宝/天猫 (Taobao / Tmall)

| 参数 | 要求 |
|------|------|
| 主图尺寸 | ≥800×800px，推荐 1200×1200px |
| 比例 | 1:1 |
| 格式 | JPG / PNG |
| 文件大小 | ≤3MB |
| 背景 | 主图必须白底，详情图可场景 |
| 数量 | 主图 5 张 + 详情图 |
| 特殊要求 | 主图禁止促销文字和水印 |

**推荐角度**：正面、左前45°、右前45°、背面、细节特写
**CLI 参数**：`--ratio 1:1`（默认即可，无需 upscale）

## 拼多多 (Pinduoduo)

| 参数 | 要求 |
|------|------|
| 轮播图尺寸 | 750×352px |
| 详情图尺寸 | 1200×1200px |
| 轮播比例 | 约 16:9 |
| 详情比例 | 1:1 |
| 格式 | JPG / PNG |
| 文件大小 | ≤1MB |
| 背景 | 白底或浅色底 |
| 数量 | 轮播 10 张 + 详情图 |

**推荐角度**：正面、左前45°、右前45°、场景图×2
**CLI 参数**：轮播 `--ratio 16:9`，详情 `--ratio 1:1`

## Amazon

| 参数 | 要求 |
|------|------|
| 主图尺寸 | ≥1600×1600px，推荐 2000×2000px |
| 比例 | 1:1 |
| 格式 | JPEG / PNG / TIFF / GIF |
| 文件大小 | ≤10MB |
| 背景 | 主图必须纯白 (RGB 255,255,255) |
| 数量 | 7-9 张 |
| 特殊要求 | 商品占画面 ≥85%，无水印/logo/文字 |

**推荐角度**：正面(主图)、背面、左侧、右侧、左前45°、俯视、细节特写、使用场景
**CLI 参数**：`--ratio 1:1`，生成后 MUST `meitu image-upscale` 确保 ≥2000px

## 平台 → CLI 参数速查

| 平台 | --ratio | 需要 upscale | 背景要求 |
|------|---------|-------------|---------|
| 淘宝/天猫 | 1:1 | 否 | 主图白底 |
| 京东 | 1:1 | 否 | 白底为主 |
| 拼多多(轮播) | 16:9 | 否 | 白/浅色底 |
| 拼多多(详情) | 1:1 | 否 | 白/浅色底 |
| Amazon | 1:1 | MUST (≥2000px) | 纯白 RGB(255,255,255) |
| 小红书 | 3:4 | 否 | 场景为主 |
| 抖音 | 1:1 | 否 | 白底或场景 |

## 通用角度优先级

不确定用哪些角度时，按此优先级选择：

| 优先级 | 角度 | 理由 |
|--------|------|------|
| 1 | 正面 | 所有平台首图，建立第一印象 |
| 2 | 前45° | 最具视觉吸引力，展示立体感 |
| 3 | 背面 | 展示全貌，完整产品信息 |
| 4 | 侧面 | 展示厚度/深度/轮廓 |
| 5 | 俯视 | 展示顶部设计/内部空间 |
| 6 | 细节特写 | 展示材质/工艺/质感 |
