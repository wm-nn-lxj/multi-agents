## 使用说明

1. 用户说平台名但未指定场景 → 使用标注"推荐"的行
2. 用户说平台名 + 场景 → 精确匹配
3. 用户说平台名但表中无该平台 → 询问目标尺寸或比例，根据平台特性推荐（横版/竖版/正方形）
