## 背景色标准

| 颜色 | RGB | Hex | 适用场景 |
|------|-----|-----|---------|
| 白色 | 255,255,255 | #FFFFFF | 身份证、护照、驾照、签证（最通用） |
| 蓝色 | 67,142,219 | #438EDB | 毕业证、工作证、职业资格证、社保卡 |
| 红色 | 255,0,0 | #FF0000 | 结婚证、党员证、团员证、暂住证 |
| 浅灰 | 245,245,245 | #F5F5F5 | 英国护照 |

### 背景色选用规则

- 用户明确指定颜色 → 用指定颜色
- 用户说"白底""蓝底""红底" → 对应标准色
- 用户未指定 → 根据规格的默认背景列推断
- 推断不出 → 默认白色（覆盖面最广）
